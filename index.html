Documentation

# Cloner le dépôt
git clone https://github.com/yasscode1234/AETHERION_OS.git
cd AETHERION_OS

# Compiler
make

# Lancer sous QEMU
make run

⚡ Nécessite x86_64-elf-gcc et qemu-system-x86_64.
