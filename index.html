À propos d’Atherion OS

Atherion OS est un système d’exploitation minimaliste, modulaire et expérimental, développé en C et assembleur, sous GPL v2. Il privilégie la clarté du code, la modularité et la stabilité d’API, avec un support QEMU pour l’expérimentation.

Documentation

Cloner & compiler

# Cloner le dépôt
git clone https://github.com/yasscode1234/AETHERION_OS.git
cd AETHERION_OS

# Compiler
make

# Lancer sous QEMU
make run

⚙ Prérequis: x86_64-elf-gcc, nasm, qemu-system-x86_64.

Notes

Le projet est en phase expérimentale. Les interfaces et composants peuvent évoluer rapidement pour améliorer la stabilité et la cohérence.

Terminal de démonstration

/usr/bin/atherion — demo
